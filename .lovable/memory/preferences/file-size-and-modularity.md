name: File size & modularity
description: Splitta filer när det går utan att de blir för stora. Inga rapporter om filstorlek.
type: preference
---
Bryt upp filer när det går naturligt — utan att de blir för stora.

**Regler:**
- Inga rapporter, kommentarer eller diskussioner om filstorlek efter varje arbete.
- Splittra om det går snyggt utan att skapa mikrofiler.
- Annars lämna filen i fred.

**Why:** Användaren vill inte höra om filstorlek hela tiden. Bara gör jobbet.

# Project Memory

## Core
- Starta dagen: snabb GPS auto-start vid träff; annars obligatorisk dialog. Visa alltid texten "Starta dagen" på knappen.
- Personalkalendern: samma person kan läggas i flera team samma dag. Drag/Add ⇒ lägger till; "remove from team" tar bara bort den teamraden.
